# shop
电子商城基于jsp,servlet,javabean开发,具有增删查改,ajax,异步.实现了下订单，实现了支付功能。后台增删查改以及添加修改商品的属性是否为新品是否为热销是否是
特价。初级入门项目，界面用户体验好。
